Card 02 论文概述

论文概述

  • 现有视觉-语言-动作（VLA）模型可解释性不足，难以精确执行自然语言中包含的空间、时序、逻辑约束，论文提出以信号时序逻辑（STL）作为高层语言理解与低层机器人执行的共享形式化接口的分层框架
  • 框架将语言指令中的隐式约束显式转化为STL规范，实现约束的预执行验证、执行时优化监控、失败后结构化重规划，提升语言条件机器人规划的精度、可靠性与可解释性
  • 研究围绕三个核心问题展开：精确语言遵循、学习策略与运动规划的自适应切换、少样本失败重规划
Card 03 核心贡献

核心贡献

  • 提出STeP分层机器人规划框架，将STL嵌入从高层任务推理到低层执行的全流程，作为System 2（高层VLM推理）与System 1（低层执行）的统一交互接口
  • 设计双执行器自适应切换机制：针对需要精确约束满足的子任务使用STL引导的模型预测控制（MPC），针对接触丰富或感知复杂的子任务使用预训练学习策略，兼顾约束满足能力与行为表达性
  • 利用STL的定量鲁棒性语义作为执行监控与失败检测信号，结合滚动历史日志为高层规划提供结构化执行上下文，支持少样本下的高效重规划
  • 在真实桌面操作平台上验证框架的有效性，相比无STL的消融基线在约束满足率和失败恢复效率上均有显著提升
Card 04 方法描述

方法描述

  • 采用双系统（System 2 / System 1）架构：System 2基于GPT-4oVLM，负责将自然语言指令分解为带STL规范的高层子任务序列，执行静态任务规范检查（JSON格式验证、技能库匹配、对象存在性校验等），并在重规划时结合历史日志和STL鲁棒性反馈修正计划
  • System 1负责低层动作生成，根据子任务类型自动选择执行器：对于可显式建模几何、时序或安全约束的子任务，使用STL鲁棒性作为优化目标的MPC求解器；对于难以解析建模的接触丰富行为，使用预训练的学习策略（如ResNet-based行为克隆策略）
  • 关键技术：将STL公式中的max/min算子用softmax平滑以支持梯度优化，使鲁棒性可作为MPC的目标函数；通过STL监控实时评估约束满足程度，自动触发执行器切换或重规划

数据集与资源

  • 核心模型：GPT-4o（用于高层VLM推理）、ResNet-based行为克隆（BC）策略（基于30条人工演示训练）

Card 06 评估与结果

评估与结果

  • 评估基准：真实桌面操作平台，测试9项涵盖逻辑、空间、时序约束的长时间跨语言条件操作任务，对比消融基线VLM-MPC-Cost（移除STL组件，System2输出直接MPC成本函数而非STL公式，历史日志存储MPC成本而非STL鲁棒性）
  • 主要评估指标：安全成功率（按约束类型分组统计）
  • 关键实验结果：

- 精确语言遵循：STeP在所有任务类别上的安全成功率均高于基线，其中时序约束类任务的提升最为显著

- 执行器切换：30次障碍物干扰试验中，系统29次检测到碰撞风险，25次切换至MPC完成绕行，22次最终完成抓取；纯BC策略28/30发生碰撞，纯MPC 24/30抓取失败

- 少样本重规划：初始失败的16次STeP试验中，12次在第二次尝试时修正失败，14次在三次内完成修正；基线VLM-MPC-Cost的18次初始失败中，仅7次在两次内修正，9次在三次内修正
