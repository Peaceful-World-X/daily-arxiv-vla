Card 02 论文概述

论文概述

  • 聚焦接触丰富机器人操作场景中，触觉增强视觉-语言-动作（VLA）策略的未来触觉预测应作用于哪个内部表示的核心问题，研究触觉预测的表示对齐性对操作性能的影响
  • 通过线性探针诊断发现VLA策略的中间动作专家层表示对未来触觉状态的可预测性最高，提出轻量级潜触觉预测器（LTP）在该层提供训练时的触觉接地信号
  • 在不改变策略推理路径的前提下，提升VLA对视觉无法感知的接触动态（如滑移、压力、装配对齐度）的建模能力，解决接触丰富操作的成功率瓶颈
Card 03 核心贡献

核心贡献

  • 提出表示对齐诊断方法：通过冻结策略的轻量线性探针分析VLA不同层级的表示，首次识别出中间动作专家特征是未来触觉状态最易预测的接地接口
  • 设计轻量级潜触觉预测器（LTP）：在诊断出的中间表示上预测紧凑的未来触觉潜变量，规避原始触觉信号的噪声、传感器特异性和校准偏差，训练时提供接地损失且不增加推理开销
  • 真实世界多任务验证：在5个接触丰富操作任务上，表示对齐的触觉grounding平均成功率74%，显著优于不对齐接口和多接口监督方案，且在SmolVLA和π₀两个不同VLA骨干上均验证有效
Card 04 方法描述

方法描述

  • 诊断阶段：冻结预训练VLA策略的所有参数，训练独立轻量线性探针分别探测VLM侧输出、各动作专家层表示对未来触觉序列的预测能力，通过探针误差确定最优接地接口
  • 训练阶段：在最优的中间动作专家表示上挂载LTP分支，使用可学习查询预测未来触觉潜序列，损失为预测潜变量与停止梯度的真实触觉潜变量的L2误差，与原生VLA动作损失加权联合训练，推理时直接移除LTP分支
  • 创新点：将未来触觉预测从通用辅助损失转化为表示对齐的接地目标，首次明确触觉预测的效益依赖于其作用的策略内部表示的匹配度，通过潜空间预测规避原始触觉的低级噪声问题
Card 05 数据集与资源

数据集与资源

  • 自建接触丰富操作真实世界数据集：包含USB Drive插入、白板擦拭、插头插入、灯泡拧下、可变形物体抓取5个任务，每个任务采集50条专家演示数据，由leader-follower遥操作同步采集RGB图像、机器人本体感知、触觉读数
  • 模型规模：基于SmolVLA（小型高效VLA模型，动作专家共16层）和π₀（流式动作专家的通用VLA模型）两个骨干网络开展实验
  • 训练超参数与资源：SmolVLA训练200k步，batch size 64，学习率1e-4余弦退火至2.5e-6，AdamW优化器；π₀采用LoRA微调，LoRA秩16，训练30k步，batch size 64，学习率1e-4余弦退火至1e-5，硬件配置未在原文中明确说明
Card 06 评估与结果

评估与结果

  • 评估环境：真实世界单臂机器人实验台，搭载ARX R5机器人和PaXini触觉传感器，策略输入包含第三人称/腕部相机RGB流、机器人本体感知、当前触觉读数
  • 主要评估指标：任务成功率，每个任务开展20次真实世界试验，按任务特定标准（如正确插入、擦拭覆盖率、无变形抓取等）判定执行成功
  • 关键实验结果：1. SmolVLA骨干下，表示对齐接地平均成功率74%，比VLM侧触觉预测（58%）高16个百分点，比最终动作状态预测（62%）高12个百分点，优于多接口监督方案（最高平均成功率60%）；2. π₀骨干下同样验证有效性，平均成功率73%，高于触觉条件化π₀（54%）及不对齐接口方案；3. 潜触觉预测显著优于原始触觉预测，Plug插入任务成功率从55%提升至80%；4. 方法在4个任务上取得最优性能，1个任务并列最优，性能增益跨任务一致
