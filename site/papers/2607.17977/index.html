Card 02 论文概述

论文概述

  • 论文介绍了 RynnBrain 1.1，一个涵盖 2B、9B 和 122B-A10B 三种规模的具身基础模型家族，采用统一的时空表征和物理 grounding 框架，支持具身感知、空间推理、定位和规划
  • 相比前代 RynnBrain 1.0，新增了接触点预测和原生 3D 定位任务，使模型表征和输出更直接对齐机器人操作需求
  • 开发了 RynnBrain-VLA，设计统一的跨具身动作空间与具身特定掩码机制，并在 Unitree G1、Astribot-S1 和 Tianji-Wuji 三种不同机器人平台上完成部署与验证
Card 03 核心贡献

核心贡献

  • 发布 RynnBrain 1.1 模型家族，在具身认知、定位和 3D grounding 等广泛基准上验证了清晰的规模扩展规律，其中 122B-A10B 模型在 VSI-Bench、MMSI 和 RefSpatial-Bench 上超越所有评估的专有及开源模型
  • 在预训练中引入接触点预测和原生 3D grounding（2B 和 9B 模型），通过紧凑的接触中心表示 (p, θ) 和多源 3D 数据（WildDet3D、FoundationPose）提升模型对机器人操作的物理对齐度
  • 提出 RynnBrain-VLA，采用统一跨具身动作空间（81 维）与具身特定掩码，实现异构机器人的联合多任务、多具身训练，无需强制对齐不兼容的低层动作空间
  • 通过实时分块（Real-Time Chunking）机制与三层跨具身/跨驱动部署框架（模型层、控制层、具身层），在真实机器人上实现低延迟平滑控制
Card 04 方法描述

方法描述

  • 基础架构：采用基于 Qwen3.5 的 decoder-only 视觉-语言架构，包含视觉编码器、视觉-语言投影器和 LLM 主干，集成 DeepStack 与 Interleaved MRoPE 以支持长上下文时空建模
  • 统一时空表征：将图像（T=1）和视频（T>1）统一编码为带时间位置信息的视觉 token 序列，使模型能通过自回归方式联合学习语义推理与空间定位
  • 物理 Grounding 输出空间：除自然语言 token 外，输出空间统一包含边界框、点、轨迹航点等离散化空间 token（坐标映射至 [0,1000] 整数），3D grounding 额外预测相机坐标系下的 9 维 3D 边界框参数（中心位置、尺寸、姿态）
  • 接触点预测：采用紧凑的接触中心表示替代传统的四角抓取框，通过视角条件渲染与投影管线将多维抓取标注转换为可靠的 2D 监督信号，最终形成 2.6M 训练样本
  • VLA 后训练：采用 flow matching 框架，将 RynnBrain 基础模型作为单流 Diffusion Transformer，输入打包后的语言指令、多相机观测、机器人状态和带噪动作块，迭代去噪预测 32 步动作块
  • 统一跨具身动作空间：将 81 维动作向量分为语义对齐的组（Arm-Joint、Arm-EEF、Gripper、Hand、Torso、Head），每个机器人通过掩码仅激活自身可用的物理维度，实现跨具身知识共享
  • Real-Time Chunking (RTC)：每 5 步触发新推理，利用前一动作块未执行部分引导新块去噪（动作引导强度 β=10.0），通过加权机制确保块边界连续性同时允许模型适应新观测
Card 05 数据集与资源

数据集与资源

  • 通用 MLLM 数据：LLaVA-OV-SI、LLaVA-Video、ShareGPT-4o-video、VideoGPT-plus、FineVideo、CinePile、ActivityNet、YouCook2、LLaVA-SFT、RynnBrain-Thinking
  • 多维度认知数据：RefCOCO、Google Refexp、Osprey-724K、DAM、VideoRefer-700K、SenseNova-SI-800K、VSI-590K、VLM-3R、EgoRe-5M、EgoTaskQA、Env-QA、QAEgo4D、RoboVQA、Robo2VLM、ShareRobot
  • 时空定位数据：ADE20K、COCOStuff、Mapillary、PACO-LVIS、PASCAL-Part、VG、RoboAfford、RefSpatial、Molmo2、FSD、MolmoAct
  • 3D Grounding 数据：WildDet3D-Essential（102,979 张图像，374K 标注）、WildDet3D-Synthetic（896,004 张图像，888K 标注）、FoundationPose（1,849K 图像）
  • 接触点数据：Grasp-Anything、Jacquard V2、GraspFactory、GraspNet-1B、GraspClutter6D（共 2.6M 训练样本）
  • 规划数据：AgibotWorld、Open X-Embodiment、Galaxea-G0
  • 模型规模：RynnBrain 1.1-2B、RynnBrain 1.1-9B、RynnBrain 1.1-122B-A10B（均基于 Qwen3.5 初始化）
  • 训练资源：采用 ZeRO、梯度检查点、专家并行与高效 token 分发等内存高效训练技术；122B-A10B 模型采用稀疏 MoE 训练
Card 06 评估与结果

评估与结果

  • 评估基准：在 VSI-Bench、MMSI、RefSpatial-Bench、RoboSpatial（Pointing/VQA/Overall）、MindCube、EmbSpatial、RynnBrain-Object/Spatial/Grounding/Area/Affordance/Trajectory 等基准上评估基础模型；在 Unitree G1（人形）、Astribot-S1（双臂）、Tianji-Wuji（灵巧手）上评估真实机器人性能
  • 评估指标：具身认知与定位任务的准确率、真实机器人的过程分数（process score）与成功率
  • 关键实验结果：

- 122B-A10B 模型在 VSI-Bench、MMSI 和 RefSpatial-Bench 上超越所有评估的专有模型（Gemini 3 Pro、GPT 5.4、Claude Sonnet 4.6）及开源模型

- 2B 和 9B 模型在加入显式 3D 监督后展现出强语言条件 3D grounding 能力，验证了原生 3D 建模的有效性

- 真实机器人实验表明，基于 RynnBrain 1.1 初始化的 VLA 策略在过程分数和成功率上均优于基于 Qwen 的策略和代表性通用 VLA（如 π₀）

- 联合多任务和多具身训练进一步提升了平均过程分数和最终成功率，优于单独微调的 per-task 策略

- 具身能力表现出非均匀缩放规律，具身预训练作为缩放的使能器，使大模型能更好地迁移到下游机器人策略
