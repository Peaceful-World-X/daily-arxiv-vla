Card 02 论文概述

论文概述

  • 针对机器人操作中Vision-Language-Action (VLA) 模型的终身学习需求，解决新技能获取与旧技能保持之间的塑性-稳定性权衡（plasticity-stability trade-off）难题
  • 指出现有VLA模型多采用静态离线训练范式，无法持续适应开放世界中的新物体、新指令和新场景，而现有终身学习方法存在权衡不足或全轨迹重放开销过高的问题
  • 提出 LifelongVLA 缓存高效终身学习框架，通过双时间尺度适配机制和随机重放策略，在有限内存下实现新技能的高效扩展与旧技能的稳健保留
Card 03 核心贡献

核心贡献

  • 提出 LifelongVLA 方法，使机器人能够持续获取新操作技能，同时保持已有技能，并通过缓存高效设计显著降低重放内存开销
  • 设计 Dual-Timescale LoRA Gating 模块，将模型适配分解为负责快速学习的短期路径和负责稳定记忆的长期路径，通过任务感知的权重级门控显式控制塑性-稳定性权衡
  • 开发 Cache-Efficient Stochastic Replay 模块，仅存储少量随机样本的stop-gradient前缀token，并在重放时重新采样扩散变量、重新计算后缀token，避免全轨迹存储
Card 04 方法描述

方法描述

  • 基于冻结的 PaliGemma 主干网络（Gemma 2B 视觉语言编码器 + Gemma 300M 连续动作解码器），采用 LoRA 进行参数高效微调
  • 双时间尺度LoRA Gating：短期LoRA路径使用当前任务损失更新以快速学习新技能；长期LoRA路径仅使用重放损失和蒸馏损失更新以巩固旧技能；共享门控基于冻结前缀特征计算上下文，在权重层面动态组合两条路径，推理时仅需单次前向传播
  • 缓存高效随机重放：采用 Reservoir采样 维护大小为M=500的 bounded replay cache，每任务仅缓存c_t=50个样本的stop-gradient前缀token及紧凑的状态-动作监督；重放时重新采样扩散时间步和噪声，利用当前模型重新计算后缀token，使单个缓存样本能生成多个随机重放实例；采用Masked new-replay目标函数，对当前任务损失和重放损失分别归一化，并引入教师模型蒸馏损失进一步稳定旧技能预测
Card 05 数据集与资源

数据集与资源

  • 使用 LIBERO benchmark 数据集，选取10个操作任务进行增量技能学习评估；真实机器人实验在 xArm 机器人上进行，包含5个任务序列
  • 模型规模：主干为冻结的 PaliGemma（含 Gemma 2B），动作解码器为 Gemma 300M；可训练参数为 49.99M（单LoRA基准）或 99.97M（完整双LoRA模型）
  • 训练配置：每任务训练10,000步（联合训练上界为40,000步）；LoRA秩r=16，缩放因子γ=16，缓存容量M=500，每任务缓存配额c_t=50，损失权重λ=1.0，β=0.1
Card 06 评估与结果

评估与结果

  • 评估环境：LIBERO 仿真基准（10个任务）和真实 xArm 机器人平台（5个任务）；对比基线包括SFT、LwF-LoRA、ER、Info-VLA、AtomicVLA
  • 主要评估指标：平均成功率（Average SR）、平均遗忘率（Average FOR）、过程级增量准确率（ACC_t）
  • 关键实验结果：LifelongVLA 达到 83.2% 的平均成功率和 11.4% 的平均遗忘率，相比最强基线ER分别提升 13.0 和 8.2 个百分点；重放内存成本降至 95.70 MiB/task（相比原始数据重放167.62 MiB/task显著降低）；在仿真任务5、9、10上分别达到96%、98%、96%的成功率，其中任务9实现0%遗忘；真实机器人评估所有5个任务成功率均超过80%
