Card 02 论文概述

论文概述

  • 针对多目标跟踪（MOT）中物体高度相似场景（如蜂群）的长期身份保持难题，提出 VLA-ReID（Video-Level Association re-ID）框架
  • 解决传统重识别（re-ID）模型的训练-推理不匹配问题：实例级查询训练与视频级全局关联推断之间的差异导致相似物体间的身份切换
  • 通过 FCAE（Frame-Common Appearance Estimation）和 CAS（Common-Appearance Suppression）机制，无需额外标注即可增强高度相似物体间的区分度
Card 03 核心贡献

核心贡献

  • 提出视频级关联建模框架，将实例级 re-ID 学习重构为轨迹特征集与当前帧检测特征集之间的全局关联学习，对齐训练与推断过程
  • 设计通用外观估计与抑制机制：FCAE 估计当前帧检测的通用外观方向，CAS 在轨迹和检测特征中抑制该方向上的公共外观分量，放大 discriminative 差异
  • 在 BEE24 数据集上系统性验证，VLA-ReID 在 HOTA、MOTA、AssR、AssA、IDF1 等指标上均优于 SOTA，身份切换次数（IDs）降低 28%
Card 04 方法描述

方法描述

  • 基于 TrackTrack 构建，仅替换其 pairwise appearance cost，保持检测器、运动模型、关联流程和轨迹管理不变
  • FCAE 使用可学习注意力池化从当前帧检测集估计单位范数通用外观方向 B，替代简单集合均值以增强自适应性
  • CAS 对轨迹特征和检测特征施加相同的可学习逐维抑制门控 α，通过减去投影到通用方向 B 上的分量并重新归一化，生成残差特征
  • 残差特征计算集合相对相似度矩阵 S = Ĝ Ẑᵀ，经 min-max 归一化转化为集合相对外观代价 C^set
  • 视频级片段训练：以目标帧为中心构建片段，联合呈现历史轨迹特征与当前帧全部检测，使用行-wise 和列-wise 交叉熵损失优化完整关联矩阵，并加入防坍缩正则化项
Card 05 数据集与资源

数据集与资源

  • 数据集：BEE24（蜜蜂跟踪数据集，包含密集、高度相似的小物体场景）
  • 外观编码器：FastReID 中的 AGW 模型，特征维度 D=2048，配备 GeM 池化
  • 检测器：YOLOX
  • 训练资源：8 块 NVIDIA RTX 3090 GPU，外观编码器预训练 120 个 epoch，片段训练 3 个 epoch
Card 06 评估与结果

评估与结果

  • 评估指标：HOTA（主要指标）、MOTA、AssR、AssA、IDF1、FP、FN、IDs、Frag
  • 在 BEE24 测试集上，VLA-ReID 达到 HOTA 49.2、MOTA 67.0、AssR 61.7、AssA 46.6、IDF1 64.9，IDs 为 437
  • 相比基线 TrackTrack（HOTA 48.1，IDs 606），HOTA 提升 +1.1，IDs 减少 28%（169 次）
  • 消融实验验证：视频级片段训练 + FCAE + CAS 的组合取得最佳性能，共同外观抑制显著提升候选排序准确率（Rank-1 从 86.6% 提升至 88.2%）
