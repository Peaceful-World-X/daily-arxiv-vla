Card 02 论文概述

论文概述

  • 针对长期机器人操作中子任务耦合问题：VLA策略在几何上成功的终端状态可能对下游任务脆弱，导致级联失败
  • 提出 Foresight Residual RL 框架，通过离线估计的前视价值(foresight value)作为奖励乘数，优化子任务交接质量而非仅优化单任务成功率
  • 在 Isaac Gym 的扳手-螺母装配任务（抓取、移动插入、旋转三阶段）上验证，全任务成功率从 54.5% 提升至 85.6%
Card 03 核心贡献

核心贡献

  • 提出 Foresight Residual RL 框架，通过前视价值函数建模子任务间的因果依赖，将局部技能执行与全局任务成功在理论上对齐
  • 为 VLA 模型提供可扩展的实践方案：离线视觉前视学习 + 反向前视归纳 (backward foresight induction)，避免联合层次优化中的非平稳性
  • 在接触丰富的长程装配任务上验证，相比标准残差强化学习提升 +31.1% 全任务成功率，同时保持单任务成功率不变
Card 04 方法描述

方法描述

  • 前视价值函数 V_{k+1:K}(s)：从阶段 k+1 初始状态 s 开始的期望折扣最终阶段成功概率，用于量化终端状态对未来成功的影响
  • 局部前视价值：将阶段 k 的终端奖励与前视价值相乘，仅修改终端奖励保留阶段内优化过程
  • 反向归纳训练：从最后阶段 K 向前训练，固定下游策略以估计前视价值，确保估计量平稳
  • 视觉前视预测器：基于 DINOv2-S/14 的双视角（第三人称+腕部相机）特征融合网络，使用二项式负对数似然训练，输出下一阶段成功概率
  • 残差策略：a_t = π₀(o_t) + α·π_res(o_t)，α=0.1，仅输入腕部相机图像，actor 输出层零初始化保证训练初期行为不变
Card 05 数据集与资源

数据集与资源

  • Isaac Gym 仿真环境，GPU 加速 PhysX 接触模拟
  • 任务：7-DoF Kuka IIWA 机器人 + 11-DoF Robotiq 三指夹爪，扳手-螺母精密装配（抓取、移动插入、旋转），5 种尺寸变体，间隙小于 2mm
  • 专家演示：每子任务 2,560 条演示，使用状态机专家（含地面真值位姿等特权信息）
  • 基础模型：微调 π₀ VLA 模型，动作块预测视界 H=20
  • 训练框架：PPO via rl_games，128 个并行环境，10 轮 critic-only 预热 + 300 轮联合训练
  • 预测器数据：N=2,560 终端状态，K_rep=5 次下游策略 rollout 标注，80/10/10 划分，LoRA rank 8 微调 DINO 主干
Card 06 评估与结果

评估与结果

  • 评估设置：512 集 episodes，4 个随机种子，报告均值±标准差
  • 评估指标：单子任务（神谕初始化）、连续两阶段链、全任务成功率
  • 全任务成功率：

- π₀ 端到端：54.9%

- π₀ 级联：41.4%

- π₀ + 残差（常数奖励）：54.5%

- π₀ + 残差 + 前视（本文）：85.6% (±3.9%)

  • 关键发现：常数奖励残差 RL 虽提升单任务成功率（插入 45.7%→92.2%），但全任务无提升，因其降低终端状态质量（抓取终端预测分数从 0.64 降至 0.32）
  • 前视预测器质量：抓取→插入边界 86.2% 准确率（基线 73.9%），插入→旋转边界 93.9% 准确率（基线 90.4%）
  • 交接质量：Grasp+Insert 连续成功率从 55.5% 提升至 87.3%，Insert+Rotate 从 83.4% 提升至 91.8%
